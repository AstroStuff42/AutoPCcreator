A Python script that lets you create complex macroses or even bots on PC with step-by-step system and GUI.

In image captioning you can set:
- Confidence (how much of image have to match for it to be considered true)
- Search timeout
- Search interval
- Convenient screenshot and crop utility

As for click options with Image Recognition:
- Click Offset (x, y)
- Button (left, right)
- Repeat count
- Repeat delay
- Click duration
- Delay before click
- Delay after click

Alongside this GUI macro/bot creator features:

- Just click (x, y, can click on screen to show where it is!)
- Delay
- Mouse scrool (Direction: up, down; Amount (lines))
- Type text (Choosen text, interval)
